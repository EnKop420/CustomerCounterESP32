void loop ( )

Main loop of the program. Runs continuously to check the WiFi connection and handle touch input.

This function first checks if the device is connected to WiFi. If not, it exits early. If connected, it reads the touch sensor value and checks if it is below the threshold. If the threshold is met, it calls the onTouch() function and sets the isTouched flag to true. If the touch value is above the threshold, it resets the isTouched flag.
